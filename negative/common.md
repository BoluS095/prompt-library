# Common Negative Prompts

Use these as a starting point and customize per project.

## Universal (works for most generations)
```
blurry, low quality, watermark, text, distorted, oversaturated,
amateur, deformed, disfigured, bad anatomy, extra limbs,
missing limbs, floating objects, disconnected, out of frame,
cropped, low resolution, jpeg artifacts, ugly, duplicate
```

## For Product Photography
```
blurry, low quality, watermark, text, distorted, unrealistic,
bad proportions, floating, disconnected parts, messy background,
cluttered, amateur lighting, harsh shadows, overexposed
```

## For Portraits
```
blurry, low quality, watermark, deformed face, bad anatomy,
extra fingers, missing fingers, asymmetric eyes, bad teeth,
cross-eyed, unnatural skin, plastic skin, uncanny valley
```

## For Social Media Graphics
```
blurry, low quality, watermark, cluttered, busy, hard to read,
poor composition, amateur, inconsistent style, distorted text
```

## For Abstract / Backgrounds
```
text, watermark, objects, people, faces, logos, brands,
busy, cluttered, noisy, banding, artifacts, harsh edges
```
